# Manufacturing Downtime Analysis & Prediction

## 📌 Project Overview
This project analyzes manufacturing line productivity and downtime patterns to support data-driven decision-making. 

The objective is to clean and preprocess production data, identify key downtime drivers, and predict next-day downtime to optimize batch production planning.

---

## 🎯 Project Objectives
- Clean and preprocess manufacturing data
- Identify key downtime factors
- Develop business-driven analytical questions
- Predict next-day downtime
- Recommend optimal batch production quantity
- Build an interactive Power BI dashboard
- Present insights in a final report and presentation

---

## 🛠 Tools & Technologies
- Power BI
- Excel
- Data Cleaning & Transformation
- DAX
- Business Intelligence & Predictive Analysis

---

## 📊 Project Phases

### Week 1 – Data Cleaning & Preprocessing
- Handle missing values
- Remove duplicates
- Data validation & transformation
- Prepare dataset for analysis

### Week 2 – Analysis Phase
- Define key business questions
- Downtime analysis by machine, shift, operator, and date
- Develop next-day downtime prediction logic
- Estimate recommended production batches

### Week 3 – Dashboard Development
- Build interactive Power BI dashboard
- Visualize KPIs:
  - Total Downtime
  - Downtime by Machine
  - Downtime by Shift
  - Downtime Trend
  - Predicted Downtime
  - Recommended Batch Count

### Week 4 – Final Presentation
- Final analytical report
- Model explanation
- Business recommendations
- Professional presentation slides
